# WeChat2048Game
微信小程序2048小游戏
游戏初始化功能
创建 4x4 空网格 ：游戏开始时，会生成一个 4x4 的空网格作为游戏的基本盘面。
添加初始方块 ：在空网格上随机添加两个方块，方块的值有 90% 的概率是 2，有 10% 的概率是 4，为游戏的开始提供初始元素。
方块操作功能
触摸滑动操作 ：通过监听触摸开始、触摸移动和触摸结束事件，实现对游戏方块的滑动操作。玩家可以在游戏界面上滑动手指，使方块按照滑动方向进行移动。
方块移动与合并 ：当玩家滑动方块时，方块会按照一定的规则进行移动和合并。具体来说，相同值的方块在移动时会合并成一个方块，其值为原来两个方块值之和。同时，方块在移动过程中会紧靠在一起，没有空隙。
添加新方块 ：每次成功移动和合并方块后，会在空白的格子中随机添加一个新的方块，保持游戏的持续性和挑战性。新方块的值同样遵循 2 和 4 的概率规则。
分数计算功能
实时计分 ：游戏会根据玩家合并的方块值来计算当前分数。每当两个方块合并时，会将合并后方块的值累加到当前分数中，使玩家能够实时了解自己的得分情况。
最高分记录 ：游戏会记录玩家在当前游戏过程中获得的最高分数，并将其与本地存储中的历史最高分进行比较。如果当前分数超过了历史最高分，则会更新最高分记录，让玩家能够看到自己在游戏中的最佳表现。
游戏状态管理功能
检测游戏结束条件 ：在游戏过程中，会不断检测游戏是否结束。当满足以下两个条件之一时，游戏结束：一是网格中没有空格子，且相邻的方块都不能合并；二是网格中出现了一个值为 2048 的方块（可选，在一些变体游戏中）。
重新开始游戏 ：无论游戏处于何种状态，玩家都可以随时重新开始一局新游戏。重新开始后，游戏会清空当前的游戏数据，重新初始化一个 4x4 的空网格，并添加两个初始方块，让玩家可以重新开始挑战。
数据存储功能
保存最高分 ：为了保持玩家的游戏成绩，游戏会将最高分保存到本地存储中。这样，即使玩家关闭游戏或重新打开游戏，最高分记录依然能够保留下来。
